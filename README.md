# Cadastro de Usuários — Projeto Fullstack

Projeto de estudo **fullstack** desenvolvido com o objetivo de praticar a integração entre **frontend** e **backend**, organização de repositório e versionamento com Git/GitHub.

O sistema permite realizar o **cadastro e listagem de usuários**, consumindo uma API desenvolvida em Node.js a partir de uma interface web em React.

---

## Tecnologias Utilizadas

### Backend
- Node.js
- Express
- Prisma ORM
- Banco de dados relacional
- JavaScript

### Frontend
- React
- Vite
- JavaScript
- CSS
